# WalletCare Front-end

Flutter-based frontend for the **WalletCare** application.

## Project Name
money_frontend

## Description
WalletCare Front-end is a Flutter application used for building the user and admin interfaces of the WalletCare personal finance platform.

## Tech Stack
- Flutter
- Dart
